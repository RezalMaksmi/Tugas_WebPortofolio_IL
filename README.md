## Installation

1. Lakukan clone pada repositori ini
   ```sh
   git clone https://github.com/RezalMaksmi/mindes.git
   ```
2. Masuk kedalam folder hacktiv8-finalproject-3.git
   ```sh
   cd mindes
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run program dengan
   ```sh
   npm start
   ```
